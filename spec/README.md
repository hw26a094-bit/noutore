## 共通システム要件（想定）
別AIモデルでの実装をスムーズに進めるための共通想定です。

- **共通ゲームシステム**:
  - **仕様（ルール）表示と開始の流れ**:
    - メインメニュー画面では、操作方法や出題範囲（仕様範囲）を表示しない。
    - プレイヤーが「挑戦する」ボタンを押した段階で、初めてそのゲームの詳細なルール説明（仕様）や操作方法が表示される。
    - 仕様説明と同じ画面上で、「通常モード（制限時間あり）」または「無制限モード（制限時間なし）」を選択してゲームを開始する。
  - **モード選択**: ゲーム開始前に以下のいずれかを選択可能。
    - **通常モード**: 制限時間あり（全体で60秒のカウントダウン。制限時間内に何点獲得できるかを競う。※Nバック記号記憶ゲーム等の一部除く）。
    - **無制限モード**: 制限時間なし（タイマーはカウントアップ、または表示なし。自分のペースで好きなだけプレイ可能）。
  - **途中終了（リザルト移行）**: ゲームプレイ中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアのまま即座にゲームを終了し、リザルト（スコア保存）画面に遷移できる。
  - **ライフ**: 無制限（ミスをしてもゲームオーバーにならず、プレイを継続できる。ただしペナルティとしてスコアが減点される）。
- **対象プラットフォーム**: Webブラウザ（PCおよびスマートフォン対応のレスポンシブWebアプリ）
- **技術スタック（推奨）**: HTML5, CSS3 (Tailwind CSS 等), JavaScript / TypeScript, React
