# Nバック記号記憶ゲーム 仕様書

## 1. 概要
画面に一定間隔で1つずつシンプルな記号（〇、✕、△、□など）が表示されます。記号が表示されるたびに、リアルタイムで「指定されたN個前に表示された記号」はどれだったかを選択肢から選んで回答していく、ワーキングメモリ（作業記憶）を極限まで鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、現在のN（「1つ前」「2つ前」「3つ前」などの表示）、現在の回答数（例：5 / 25）
- **中央部（メイン記号表示領域）**:
  - 現在表示された最新の記号が大きく中央にカードのように提示される。
- **下部（回答ボタン領域）**:
  - 選択肢ボタン（例：[ 〇 ] [ ✕ ] [ △ ] [ □ ]）
  - マウス操作ホバーエフェクト付き。

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（難易度の変化やN数の上昇条件、仕様範囲）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法やルール仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし / 全25問回答完了まで）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。最初の記号が順番に表示され始める。
  3. 最初のN個（例：1-backなら1個、2-backなら2個）が表示される間は、前回の記号が存在しないため、覚えるフェーズとなる。
  4. 規定数（N個）が表示された後、次の新しい記号が表示されると同時に「N個前に表示された記号は何だったか？」を回答するフェーズが開始される。
  5. 新しい記号が表示されるたびに、プレイヤーは下部のボタンから「N個前の記号」を選択してクリック回答する。
  6. 通常モードでは60秒が経過（タイムアップ）した時点でプレイは終了となる。無制限モードでは全25問の回答を終えた時点でゲーム終了（リザルト画面へ）。
  7. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
- **リアルタイム難易度（N）の上昇**:
  - 回答を重ねる（正解数を連続で増やすなど）ことで、プレイ中にリアルタイムでNの数が変動・上昇します。
    - **1-back（1つ前を回答）**: 開始時〜
    - **2-back（2つ前を回答）**: 正解を一定数重ねると自動的に2つ前へと切り替わる。
    - **3-back（3つ前を回答）**: さらに正解を重ねると自動的に3つ前へと切り替わる。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲームスタートから終了まで全体で60秒。
  - 無制限モード：制限時間なし（全25問の回答が完了するまで）。
  - ライフは無制限（回答ミスをしてもゲームオーバーにならず、プレイは強制終了しない）。
- **操作仕様**:
  - **マウス操作のみ**: 画面上の記号選択ボタンを直接クリックして回答します。このゲームにおいて、キーボード操作は使用しません。
- **記号の種類**:
  - シンプルな4種類の図形： 〇（丸）、✕（バツ）、△（三角）、□（四角）

## 4. スコアシステム（仕様）
- **正解**: +200点（現在のNの数に応じてさらにボーナス加算あり）。
- **不正解（ミス）**: ライフの減少はなく、ペナルティとして保有スコアから減点（例：-80点。ただしスコアの最小値は0点）され、次の記号表示へ進む。
- **ゲーム終了**: 合計25問の回答が完了した時点でゲームが終了し、最終獲得スコアが記録として残る。
