# 人数数えゲーム 仕様書

## 1. 概要
画面に表示される「家」に対して、人や動物などのキャラクターが出入りします。プレイヤーは出入りする人数を頭の中で追跡し、最終的に「家に残っている人の数」を回答する、注意力と認知力を鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、制限時間（通常モード：60秒のカウントダウン / 無制限モード：なし、または回答数表示）、現在の難易度/レベル、途中終了ボタン（「終了してリザルトへ」）
- **中央部（メイングラフィック領域）**:
  - 中央に「家（グラフィックまたはイラスト）」を配置。
  - 左側から家の中にキャラクターが入るアニメーション、家から右側へキャラクターが出ていくアニメーションが表示される。
- **下部（回答入力領域）**:
  - デジタルテンキー（0〜9、クリア、決定ボタン）。
  - キーボードを使わない、あるいはマウスでプレイする場合のクリック用。

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（難易度の変化やノイズの出現条件、仕様範囲）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法やルール仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。
  3. ゲームが開始されると、家の中に初期人数（0人）がいる状態からスタート。
  3. キャラクターが家に入ったり（加算）、出たり（減算）するアニメーションが指定ステップ数分繰り返される。
     - **【重要ルール】**: 最初（レベル1）は、家からキャラクターが出て行くことはなく、入る動作のみが発生します。
  4. 出入りがすべて終了した後、回答キーパッドが活性化し、プレイヤーが数値を入力して決定ボタンを押す。
  5. 即座に正誤判定が行われ、次の問題（セット）が即座に開始される。
  6. 通常モードでは60秒が経過（タイムアップ）した時点でプレイは終了となる。無制限モードではプレイヤーが終了ボタンを押すまで無限にプレイが続く。
  7. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲームスタートから終了まで全体で60秒。
  - 無制限モード：制限時間なし。
  - ライフは無制限（回答を誤っても、ゲームオーバーで強制終了することなくプレイを継続できる）。
- **同時出入り時の表示改善（重なり防止）**:
  - 複数人が同時に出入りする際、キャラクター同士が重なって見えなくならないよう、以下の対策を適用します：
    - 各キャラクターの登場タイミングをずらす（ミリ秒単位のスタッガーディレイ）。
    - 縦方向の表示位置（Y座標オフセット）をキャラクターごとにずらして表示。
- **PC・キーボード操作仕様**:
  - **マウス操作**: 画面に提示されたデジタルテンキーの各ボタンを直接クリックして数値を入力し、「決定」ボタンをクリック。
  - **キーボード操作**:
    - **「数字キー（0〜9）」および「テンキー（0〜9）」**: 残り人数の数値を直接タイピング入力。
    - **「Backspace」または「Delete」キー**: 入力中の数値をクリア（1文字消去）。
    - **「Enter」キー**: 入力した計算結果の確定と正誤判定の送信（決定ボタン押下と同じ挙動）。
- **難易度に応じた自動拡張（動的仕様）**:
  - レベル（難易度）が上がるにつれて、以下の要素が変化する：
    1. **移動スピード**: キャラクターが出入りする移動アニメーションの速度が高速化。
    2. **同時移動人数**: 一度に（同時に）家を出入りするキャラクターの最大数が増加（例：最初は1人ずつ、高レベルでは複数人が同時に出入りする）。
    3. **出入りステップ数**: 1つの問題の中で、出入りが発生する合計回数（ステップ数）が増加。
    4. **人以外の生き物（動物など）の出現（おとり要素）**:
       - 人以外のキャラクター（犬、猫など）も家に出入りするようになります。
       - **【重要ルール】**: 動物などの人以外のキャラクターは**カウント対象外**（ノイズ・おとり）です。プレイヤーは人以外の出入りを完全に無視し、**「人間の人数のみ」**を正確に追跡する必要があります（例：人が2人入り、犬が1匹入った場合、家の中にいるカウントは「2人」のまま）。

## 4. スコアシステム（仕様）
- **正解（クリア）**: 難易度やステップ数、同時移動数に応じたポイント加算。
- **不正解（ミス）**: ライフの減少はなく、ペナルティとして保有スコアから減点（スコアの最小値は0点）。
- **ゲーム終了**: 通常モードでは60秒が経過した時点、無制限モードではプレイヤーが「終了してリザルトへ」ボタンを押した時点で、そのプレイにおける最終スコアが記録として残る。
