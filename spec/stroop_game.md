# ストループ判定ゲーム 仕様書

## 1. 概要
画面に「表示される言葉（意味）」と「文字の色」が不一致な状態でテキストが表示されます。指定されたルールに従い、プレイヤーが正しい回答を瞬時に選択する、反射神経と判断力を鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、制限時間（通常モード：60秒のカウントダウン / 無制限モード：なし、または回答数表示）、現在のコンボ数、途中終了ボタン（「終了してリザルトへ」）
- **中央部（インストラクション領域）**:
  - 現在のルール指示表示。「意味を答えろ！」または「色を答えろ！」
  - ルール指示は目立つ色やフォントで表示。
- **中央部（メインお題領域）**:
  - お題ワードテキスト（例：赤色のインクで大きく「あお」と表示）
- **下部（回答ボタン領域）**:
  - 選択肢ボタン（例：[あか] [あお] [みどり] [きいろ]）
  - マウスカーソルを合わせた際、どのボタンを指しているかが視覚的に直感で分かるよう、ホバーエフェクト（ボタンの色の変化や一回り大きくなる演出など）が設定される。

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（色や指示などの仕様範囲）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法やルール仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。
  3. ゲームが開始されると、ルール（「意味」か「色」か）とお題ワード（「言葉」と「インク色」の組み合わせ）がランダムに提示される。
  3. プレイヤーは下部のボタンから正しい答えを選択する。
  4. 回答後、即座に「正解」または「不正解」が判定され、次の問題が提示される。
  5. 通常モードでは60秒が経過（タイムアップ）した時点でプレイは終了となる。無制限モードではプレイヤーが終了ボタンを押すまで無限にプレイが続く。
  6. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲームスタートから終了まで全体で60秒。
  - 無制限モード：制限時間なし。
  - ライフは無制限（回答ミスをしてもゲームオーバーにならず、プレイは強制終了しない）。
- **操作仕様（重要）**:
  - **マウス操作のみ**: 画面上の選択肢ボタンを直接クリックして回答します。このゲームにおいて、キーボード操作は使用しません。
- **問題のランダム作成ルール**:
  - 対象とする色：赤（あか）、青（あお）、緑（みどり）、黄（きいろ）
  - 表示する「言葉」と「文字の色」はランダム。
  - ルール指示（「意味」か「色」か）もランダムに切り替わる（あるいは、一定問題ごとに切り替わる）。

## 4. スコアシステム（仕様）
- **正解**: +100点 × コンボ倍率
- **不正解（ミス）**: コンボが途切れ、スコアからペナルティとしての減点（例：-50点。ただしスコアの最小値は0点）
- **コンボボーナス**: 連続で正解するごとに倍率や追加点が発生。
- **ゲーム終了**: 通常モードでは60秒が経過した時点、無制限モードではプレイヤーが「終了してリザルトへ」ボタンを押した時点で、そのプレイにおける最終スコアが記録として残る。
