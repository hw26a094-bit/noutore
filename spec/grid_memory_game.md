# 瞬間グリッド記憶ゲーム 仕様書

## 1. 概要
グリッド（格子）状に配置されたパネルの一部が順番、または同時に一瞬だけ発光します。プレイヤーはその光ったパターンを記憶し、同じパネルを同じ順序（または同じ位置）でタップして再現する、空間記憶力と順序記憶力を鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、制限時間（通常モード：60秒のカウントダウン / 無制限モード：なし、または回答数表示）、現在のレベル（例：Lv.3）、途中終了ボタン（「終了してリザルトへ」）
- **中央部（メイングリッド領域）**:
  - 正方形のグリッド（難易度に応じて 3x3、4x4、5x5 と可変）
  - 各パネルはデフォルトで消灯状態。
  - 发光時には明るい色に変化し、段階に応じた異なる音（音階）を伴う。
- **下部（操作説明・アクション領域）**:
  - 状態インジケータ（「光る順番を覚えてください！」・「順番通りにタップしてください！」）

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（グリッドの拡張条件や難易度の仕様範囲）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法やルール仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。
  3. ゲーム開始時に、現在のレベルに応じた回数だけパネルが「順番（1箇所ずつ）」に点灯・発光する。
  3. 全ての発光演出が終了すると「プレイヤーのターン」になり、グリッドのタップが可能になる。
  4. プレイヤーは**「光った順番（順序）」を完全に守って**同じパネルを連続でタップする。
  5. 全て正しい順番で再現タップできればレベルクリア。次のレベル（または記憶枚数が+1増えた状態）へ進行する。
  6. 順番を間違えた時点で、その問題は即時に「失敗」となり、スコアが減点され、次の新しい問題（またはリトライ）へ移行する。
  7. 通常モードでは60秒が経過（タイムアップ）した時点でプレイは終了となる。無制限モードではプレイヤーが終了ボタンを押すまで無限にプレイが続く。
  8. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲームスタートから終了まで全体で60秒。
  - 無制限モード：制限時間なし。
  - ライフは無制限（間違えてもゲームオーバーにならず、制限時間が切れるまでプレイし続けることができる）。
- **PC・キーボード操作仕様（重要）**:
  - **マウス操作**: 光った順に、直接グリッド上の各パネルをクリックしてタップ再現。
  - **キーボード操作**:
    - グリッドのいずれか一つのパネルに「フォーカス枠線」が表示される。
    - **「矢印キー（上下左右 ◀ / ▶ / ▲ / ▼）」**: フォーカスするターゲットパネルを選択移動。
      - 上 鍵：上のグリッドへ移動
      - 下 鍵：下のグリッドへ移動
      - 左 鍵：左のグリッドへ移動
      - 右 鍵：右のグリッドへ移動
    - **「Enter」キーまたは「Space」キー**: 現在フォーカスが当たっているグリッドのパネルをタップ（決定）する。
- **グリッドサイズと難易度の自動拡張**:
  - レベル（難易度）が上がるにつれて、グリッドの物理サイズも段階的に拡張する。
    - **レベル 1 〜 4**: 3x3 グリッド（点灯枚数：3枚 〜 4枚）
    - **レベル 5 〜 8**: 4x4 グリッド（点灯枚数：4枚 〜 5枚）
    - **レベル 9 〜**: 5x5 グリッド（点灯枚数：5枚 〜 6枚）
  - ※レベルが上がるほど、表示される個数が増え、テンポも高速化する。

## 4. スコアシステム（仕様）
- **完全再現成功（レベルクリア）**: レベルの難易度に応じたポイント加算、および連続正解（コンボ）ボーナスが発生する。
- **再現失敗（ミス）**: ライフの減少はなし。ペナルティとして保有スコアから減点（例：-100点。ただしスコアの最小値は0点）される。
- **ゲーム終了**: 通常モードでは60秒が経過した時点、無制限モードではプレイヤーが「終了してリザルトへ」ボタンを押した時点で、そのプレイにおける最終スコアが記録として残る。
