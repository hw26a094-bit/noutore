# 限界暗算ゲーム 仕様書

## 1. 概要
画面に数値や演算子が組み合わさった計算式が最初からすべて表示されます。プレイヤーは暗算で計算し、最終的な合計値を回答する、計算力を鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、制限時間（通常モード：60秒のカウントダウン / 無制限モード：なし、または経過時間表示）、途中終了ボタン（「終了してリザルトへ」）
- **中央部（メイン表示領域）**:
  - 計算式全体が**最初から画面中央にすべて表示される**。
  - 例：[ 5 + 12 - 2 ]
- **下部（回答入力領域）**:
  - デジタルテンキー（0〜9、マイナス符号、クリア、決定ボタン）
  - キーボードを使わない、あるいはマウスでプレイする場合のクリック用。

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（出題範囲・桁数など）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法や出題仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。
  3. ゲームが開始されると、完成した計算式が最初からすべて表示された状態になる。
  3. プレイヤーは数値を計算し、回答キーパッドを使って数値を入力して決定ボタンを押す。**【計算の優先順位】**: 掛け算（×）と割り算（÷）は、足し算（+）と引き算（-）よりも優先して計算します。
  4. 即座に正誤判定が行われ、次の問題が即座に開始される。
  5. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
  6. 通常モードでは60秒が経過（タイムアップ）した時点でプレイは終了となる。無制限モードではプレイヤーが終了ボタンを押すまで無限にプレイが続く。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲーム全体で60秒。
  - 無制限モード：制限時間なし。
  - ライフは無制限（回答を誤っても、ゲームオーバーで強制終了することなくプレイを継続できる）。
- **PC・キーボード操作仕様（重要）**:
  - **マウス操作**: 画面に提示されたデジタルテンキーの各ボタンを直接クリックして数値を入力し、「決定」ボタンをクリック。
  - **キーボード操作**:
    - **「数字キー（0〜9）」および「テンキー（0〜9）」**: 計算結果の数値を直接タイピング入力。
    - **「マイナス（-）キー」**: 答えがマイナス値になった場合の符号入力。
    - **「Backspace」または「Delete」キー**: 入力中の数値をクリア（1文字消去）。
    - **「Enter」キー**: 入力した計算結果の確定と正誤判定の送信（決定ボタン押下と同じ挙動）。
- **問題の動的生成**:
  - 四則演算（足し算・引き算・掛け算・割り算）がランダムに登場する。
  - **桁数の設計基準**:
    - **足し算（+）・引き算（-）**: 1桁〜**最大3桁**まで登場
    - **掛け算（×）・割り算（÷）**: 1桁〜**最大2桁**まで登場
    - ※割り算（÷）の演算の際は、必ず割り切れて答えが整数になるような数値ペアが自動生成される。

## 4. スコアシステム（仕様）
- **正答（クリア）**: 難易度やステップ数、演算子の複雑さに応じたポイント（例：+500点）が加算される。
- **誤答（ミス）**: ライフの減少はなく、ペナルティとして保有スコアから減点（例：-200点。ただしスコアの最小値は0点）される。次の問題へ進む。
- **ゲーム終了**: 通常モードでは60秒が経過した時点、無制限モードではプレイヤーが「終了してリザルトへ」ボタンを押した時点で、そのプレイにおける最終スコアが記録として残る。
