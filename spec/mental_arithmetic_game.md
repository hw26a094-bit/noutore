# 限界暗算ゲーム 仕様書

## 1. 概要
画面に数値や演算子が1つずつ一定間隔で表示されます。プレイヤーは前に表示された数値を記憶し、頭の中で計算を続けながら、最終的に表示が終了した時点での合計値を回答する、計算力と一時記憶（ワーキングメモリ）を鍛える脳トレゲームです。

## 2. 画面レイアウト案
- **ヘッダー部**: スコア、制限時間（全体：60秒のカウントダウン）、問題のステップ進行度（例：ステップ 3/5）
- **中央部（メイン表示領域）**:
  - 数値や演算子が1枚ずつカードのように大きく中央に表示される（フェードイン・アウトのエフェクト）。
  - 例：[ 5 ] -> [ × 12 ] -> [ -2 ] -> [ 終了。答えは？ ]
- **下部（回答入力領域）**:
  - デジタルテンキー（0〜9、マイナス符号、クリア、決定ボタン）
  - キーボードを使わない、あるいはマウスでプレイする場合のクリック用。

## 3. ゲームルール＆仕様
- **基本動作フロー**:
  1. ゲーム開始ボタンを押すと、60秒のカウントダウンタイマーが開始される。
  2. ゲームは制限時間内（60秒）で何問解けるかを競うスコアアタック形式。
  3. 問題（1セット）が開始されると、設定されたステップ数（例：3〜5ステップ）に合わせて、順番に数値・演算子が表示される。
  4. 表示される間隔は一定時間（例：1秒ごと）。
  5. すべての表示が終わった後、回答キーパッドが活性化し、プレイヤーが数値を入力して決定ボタンを押す。
  6. 即座に正誤判定が行われ、次の問題（セット）が即座に開始される。
  7. 60秒が経過（タイムアップ）した時点でプレイは終了となる。
- **制限時間・ライフに関するルール**:
  - 制限時間はゲーム全体で60秒。
  - ライフは無制限（回答を誤っても、ゲームオーバーで強制終了することなく次の問題に挑戦できる）。
- **PC・キーボード操作仕様（重要）**:
  - **マウス操作**: 画面に提示されたデジタルテンキーの各ボタンを直接クリックして数値を入力し、「決定」ボタンをクリック。
  - **キーボード操作**:
    - **「数字キー（0〜9）」および「テンキー（0〜9）」**: 計算結果の数値を直接タイピング入力。
    - **「マイナス（-）キー」**: 答えがマイナス値になった場合の符号入力。
    - **「Backspace」または「Delete」キー**: 入力中の数値をクリア（1文字消去）。
    - **「Enter」キー**: 入力した計算結果の確定と正誤判定の送信（決定ボタン押下と同じ挙動）。
- **問題の動的生成**:
  - 四則演算（足し算・引き算・掛け算・割り算）がランダムに登場する。
  - **桁数の設計基準**:
    - **足し算（+）・引き算（-）**: 1桁〜**最大3桁**まで登場
    - **掛け算（×）・割り算（÷）**: 1桁〜**最大2桁**まで登場
    - ※割り算（÷）の演算の際は、必ず割り切れて答えが整数になるような数値ペアが自動生成される。

## 4. スコアシステム（仕様）
- **正答（クリア）**: 難易度やステップ数、演算子の複雑さに応じたポイント（例：+500点）が加算される。
- **誤答（ミス）**: ライフの減少はなく、ペナルティとして保有スコアから減点（例：-200点。ただしスコアの最小値は0点）される。次の問題へ進む。
- **ゲーム終了**: 60秒が経過した時点で、そのプレイにおける最終スコアが記録として残る。
