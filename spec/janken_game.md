# 後出しジャンケンゲーム 仕様書

## 1. 概要
- **ゲーム名**: 後出しジャンケン
- **ジャンル**: 反射神経・判断力系
- **ゲーム概要**: 
  画面に提示される「相手の手（絵文字）」と「指示（勝つ・負ける・あいこ）」を瞬時に判断し、指示通りになる手（絵文字）を素早く選択する脳トレゲーム。

## 2. 画面レイアウト＆構成要素
- **ヘッダー領域**:
  - タイマー表示（通常モード：60秒からのカウントダウン / 無制限モード：カウントアップまたは非表示）
  - 現在のスコア表示
  - 正解数 / 連続正解（コンボ）数表示
  - 「終了してリザルトへ」ボタン
- **メイン表示領域**:
  - **相手の手（絵文字）**: ✊ (グー) / ✋ (パー) / ✌ (チョキ) を大きく中央に表示
  - **指示テキスト**: 
    - 「勝ってください」
    - 「負けてください」
    - 「あいこ（引き分け）にしてください」
- **回答操作領域**:
  - プレイヤーの選択肢ボタン。以下の絵文字ボタンを横並びで配置。
    - ✊ (グー)
    - ✌ (チョキ)
    - ✋ (パー)
  - マウスホバー時に視覚的なフィードバック（ボタン拡大など）を提供。

## 3. ゲームルール＆仕様
- **仕様表示のタイミング（重要）**:
  - メインメニュー画面では、本ゲームの操作方法や仕様（難易度の変化や手の仕様範囲）は一切表示しない。
  - プレイヤーがメインメニューの「挑戦する」ボタンを押した後に表示される「ルール仕様説明・モード選択画面」にて、初めて詳細な操作方法やルール仕様（仕様範囲）が表示される。
- **基本動作フロー**:
  1. メインメニューで「挑戦する」ボタンを押すと、本ゲームのルール仕様説明および「通常モード（制限時間60秒）」または「無制限モード（制限時間なし）」のモード選択画面が表示される。
  2. プレイヤーがいずれかのモードを選択すると、ゲームがスタートする。
  3. ゲーム開始と同時に、ランダムな「相手の手」と「指示」が表示される。
  4. プレイヤーは下部の選択肢（✊ / ✌ / ✋）から、指示に適合する正しい手をタップまたはクリックして回答する。
  5. 回答後、即座に正誤判定（「正解！」または「不正解」）が行われ、次の問題が提示される。
  6. ゲーム中、いつでも「終了してリザルトへ」ボタンを押すことで、その時点のスコアでゲームを終了し、リザルト画面に遷移できる。
- **正誤条件**:
  - **「勝ってください」の場合**:
    - 相手 ✊ に対して ✋ ＝ 正解
    - 相手 ✌ に対して ✊ ＝ 正解
    - 相手 ✋ に対して ✌ ＝ 正解
  - **「負けてください」の場合**:
    - 相手 ✊ に対して ✌ ＝ 正解
    - 相手 ✌ に対して ✋ ＝ 正解
    - 相手 ✋ に対して ✊ ＝ 正解
  - **「あいこ（引き分け）にしてください」の場合**:
    - 相手 ✊ に対して ✊ ＝ 正解
    - 相手 ✌ に対して ✌ ＝ 正解
    - 相手 ✋ に対して ✋ ＝ 正解
- **難易度変化の仕様範囲**:
  - 連続で正解を重ねる（コンボ）と、相手の手や指示テキストの切り替わりアニメーションが高速化する、または表示エフェクトが変化するなどの演出が入る。
- **制限時間・ライフに関するルール**:
  - 通常モード：制限時間はゲームスタートから終了まで全体で60秒。
  - 無制限モード：制限時間なし。
  - ライフは無制限（回答ミスをしてもゲームオーバーにならず、プレイは強制終了しない。ただしペナルティとしてスコアが減算される）。
- **スコアシステム**:
  - 正解：+10点（コンボ数に応じてボーナス加算あり）
  - 不正解：-5点（スコアは0未満にはならない）
- **操作仕様**:
  - **マウス操作のみ**: 画面上の手（✊ / ✌ / ✋）のボタンを直接クリックして回答します。このゲームにおいて、キーボード操作は使用しません。
